Documentação – Projeto Web Bootstrap 5
Tema escolhido:
Fan Club da cantora Sabrina Carpenter.

Link do projeto publicado no Github Pages: https://kaukss.github.io/ap2-front/

Explicação da estrutura do projeto:
O projeto foi desenvolvido utilizando HTML5, CSS e Bootstrap 5. As páginas foram organizadas em diferentes
arquivos HTML para melhorar a navegação e a organização do site. O sistema possui páginas de login, cadastro,
home, álbuns, shows e merch.

Componentes Bootstrap utilizados:
• Navbar
• Carousel
• Cards
• Dropdown
• Flexbox
• Grid System
• Botões responsivos
• Collapse Menu

Decisões de layout e design:
Foi utilizada uma paleta de cores suaves inspirada na estética visual da cantora, com tons pastel como azul claro,
rosa e amarelo claro. Também foram utilizadas fontes personalizadas para deixar o site mais elegante e
moderno.

Utilização do display flex:
O display flex foi utilizado para organizar os elementos da navbar, os cards de conteúdo e diferentes seções da
página, garantindo alinhamento e responsividade.

Observações relevantes sobre o desenvolvimento:
O projeto foi desenvolvido com foco em responsividade para dispositivos mobile, tablet e desktop. Também foi
priorizada a organização do código e o uso de tags semânticas do HTML5.

Utilização da IA:
Fiz a utilização do chat gpt para auxilio na escolha de cores e na estilização das outras abas mantendo o padrao da pagina inicial como fontes cores e tudo mais
justamente para pouca tempo e usar como uma ferramenta de suporte.
